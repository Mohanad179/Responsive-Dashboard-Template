# Responsive Dashboard Template

A fully responsive dashboard template built using pure HTML and CSS.

This project was built to practice creating modern responsive layouts using Flexbox and CSS Grid without relying on frameworks like Bootstrap or Tailwind.

I also created a small handmade utility system/framework for reusable shortcut classes to better understand how CSS frameworks work behind the scenes.

## 🚀 Live Demo

[View Live Demo](https://mohanad179.github.io/Responsive-Dashboard-Template/)


## ✨ Features

- Fully responsive on all screen sizes
- Built with pure HTML & CSS only
- Flexbox & CSS Grid layout system
- Custom reusable utility classes
- Dashboard cards, tables, widgets and stats sections
- Clean modern UI


## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries
